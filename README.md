# Projeto Clash Royale (Java + JavaFX)

Este projeto é uma recriação simplificada do **Clash Royale**, desenvolvida inteiramente em **Java** utilizando a biblioteca **JavaFX**. O sistema permite gerenciar cartas e decks, incluindo criação, edição, exclusão e visualização.

---

## 🏗️ Funcionalidades do Projeto

### 🃏 **Gerenciamento de Cartas**

* Adicionar novas cartas
* Editar cartas existentes
* Excluir cartas
* Associar imagens às cartas 
* Definir atributos como nome, raridade, tipo, dano etc.

### 📦 **Gerenciamento de Decks**

* Criar novos decks
* Adicionar cartas a um deck
* Remover cartas de um deck
* Visualizar decks
* Excluir decks prontos
* Calcular o custo de elixir médio do deck

### 🖼️ **Sistema de Imagem**

* Upload de imagens
* Salvamento em diretórios locais
* Exibição usando JavaFX
* Validação de formatos aceitos

---

## 🔧 Tecnologias Utilizadas

* **Java 25**
* **JavaFX 22**
